Running project Laravel 12 "CMLabs Test Front End Fulltime"

1. Required installed PHP 8.2 (Minimum)
2. Required installed Composer
3. Required installed Database Control Pannel (Xampp, Laragon, Docker, etc)
4. Run "composer install" to install all Laravel libraries
5. Duplicate .env file of Laravel and place it to root folder
6. Generate application key "php artisan key:generate"
7. Open your .env file and configure your DB_CONNECTION with your database
8. Migrate your database "php artisan migrate"
9. Run project "php artisan serve", normally it will run with "http://127.0.0.1:8000" URL
